> 结论: skyline 渲染目前还是一坨答辩


二度尝试 skyline 渲染,最终仍是以告终, 虽然官方示例的动画和自定义路由很让人心动,但是为了获得这个特性所付出的成本是巨大的.



- 各种 css 不兼容, 复杂 css 实现成本巨大, 如果你的项目有比较复杂的 css 或者使用了比较多高阶的用法, 建议不要尝试 skyline
- 开发者工具 bug 很多, 比如不支持热更新, 各种奇怪的 warning 等等...


再观望个两三年看看吧...!!!
